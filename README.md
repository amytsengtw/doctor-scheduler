# 🏥 婦產科雙軌排班系統 (v3.2 統計視覺化版)

本系統具備**雙軌排班 (Big/Small Shift)**、**雙方案決策 (Rule/VS Priority)**，並新增了**視覺化日曆**與**即時班數統計**功能。

## ✨ v3.2 新增功能

### 1. 📊 即時統計儀表板 (Statistics Dashboard)
排班完成後，系統會自動計算並顯示每位醫師的：
*   **Total**：總班數
*   **平日**：週一至週五的班數
*   **假日**：週六、週日的班數
*   *用途：總醫師可直接截圖此表格，證明班表公平性。*

### 2. 🎨 個人專屬色彩 (Color-Coded Calendar)
日曆不再只有單調的顏色。
*   系統會根據醫師名字，自動分配一個柔和的背景色 (Pastel Color)。
*   同一位醫師在大班與小班的日曆中顏色固定，方便一眼識別。

---

## 🛠 核心功能回顧

### 1. 🛡️ 絕對有解 (Always Feasible)
系統允許打破規則來換取班表產出，並將犧牲內容記錄在報告中。

### 2. ⚖️ 雙方案決策
*   **方案 A (規則優先)**：寧願犧牲 VS，也要保護 Intern/PGY 工時與 R 的意願。
*   **方案 B (主治優先)**：寧願讓 Intern/PGY 超時，也要保護 VS 的休假。

### 3. 📂 JSON 設定檔
支援一鍵下載/上傳設定，保留所有人員名單與意願。

## 🚀 使用說明

1.  **輸入資料**：填寫人員名單、指定值班與不想值班日期。
2.  **暴力運算**：點擊按鈕生成兩個方案。
3.  **檢視統計**：查看上方生成的統計表格，確認每人班數是否平均。
4.  **查看日曆**：透過顏色快速檢查是否有連續值班或分佈不均。
5.  **下載 CSV**：選擇滿意的方案下載。
