# 🏥 婦產科雙軌排班系統 (v3.5 請假管理版)

本系統區分了「絕對無法值班」與「不想值班」的層級，確保婚假、喪假或未到職等情況絕對不會被排入班表。

## ✨ v3.5 新增功能

### 1. ⛔️ 絕對排除 (Hard Constraints)
*   新增 **「請假/未到職」** 設定區。
*   在此區勾選的日期，系統**絕對不會**排入值班。
*   *注意：若設定過多導致某天完全無人可值，系統會回報「無解」。*

### 2. 🛡️ 雙層意願系統
1.  **Level 1: 絕對請假 (Absolute Leave)**：物理上無法值班 (婚假/出國/未到職)。
2.  **Level 2: 不想值班 (No-Go)**：意願上不想值班，但若為了公平性或填補人力空缺，系統仍可能排入 (會顯示在犧牲報告中)。

---

## 🛠 核心功能

### 1. 🔢 多方案生成
可生成 1~5 種不同的可行班表，供總醫師挑選最合適的方案。

### 2. ⚖️ 公平性優先
演算法核心目標是讓每位醫師的 **平日班** 與 **假日班** 數量盡量一致。

### 3. 🎨 視覺化與統計
*   **個人色彩**：自動為每位醫師分配專屬背景色。
*   **即時統計**：排班後立即顯示 loading 統計表。

## 🚀 使用說明

1.  **輸入名單**：填寫四類醫師名單。
2.  **設定請假 (最重要)**：在紅色區塊勾選「絕對無法值班」的日子。
3.  **設定意願**：在下方區塊勾選「指定值班」或「不想值班」。
4.  **運算**：選擇方案數量 (建議 3)，點擊開始。
5.  **決策**：比較不同方案的統計數據與犧牲報告，下載 CSV。
